# How to Sync VSCode with Cursor

## Cursor is Built on VSCode!

Good news: **Cursor is based on VSCode**, so most VSCode features work in Cursor automatically!

---

## Method 1: Import VSCode Settings (Easiest)

### Step 1: Find Your VSCode Settings
- **Windows Location:**
  - `C:\Users\YOUR_USERNAME\AppData\Roaming\Code\User\settings.json`
  - Or: Press `Ctrl+Shift+P` in VSCode â†’ "Preferences: Open User Settings (JSON)"

### Step 2: Copy Settings to Cursor
- **Cursor Settings Location:**
  - `C:\Users\YOUR_USERNAME\AppData\Roaming\Cursor\User\settings.json`
  - Or: Press `Ctrl+Shift+P` in Cursor â†’ "Preferences: Open User Settings (JSON)"

### Step 3: Copy & Paste
1. Open VSCode settings.json
2. Copy all content
3. Open Cursor settings.json
4. Paste and save

---

## Method 2: Install VSCode Extensions in Cursor

### Most VSCode Extensions Work in Cursor!

1. **Open Extensions in Cursor:**
   - Press `Ctrl+Shift+X` (same as VSCode!)
   - Or click the Extensions icon in sidebar

2. **Install Your Favorite Extensions:**
   - Search for the same extensions you use in VSCode
   - Click "Install"
   - Most work perfectly!

## Method 3: Sync Extensions Automatically

### Option A: Settings Sync (Built-in)
1. In VSCode: Sign in with GitHub/Microsoft account
2. Enable Settings Sync
3. In Cursor: Sign in with same account
4. Enable Settings Sync
5. Your extensions and settings will sync!

### Option B: Extension Sync Extension
1. Install "Settings Sync" extension in VSCode
2. Export your settings (creates a Gist)
3. Install same extension in Cursor
4. Import from Gist

---

## Method 4: Copy Keybindings

### VSCode Keybindings Location:
- `C:\Users\YOUR_USERNAME\AppData\Roaming\Code\User\keybindings.json`

### Cursor Keybindings Location:
- `C:\Users\YOUR_USERNAME\AppData\Roaming\Cursor\User\keybindings.json`

**Just copy the file contents!**
